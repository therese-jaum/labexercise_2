Formatting a Table

HTML Structure

Tags

  • table: table
  • table: table title
  • thread: table header
  • tbody: table content
  • trRow of a table
  • thTitle cell of a table
  • tdCell of a table

Attributes

  • colspan defines the number of columns to be merged
  • rowspan defines the number of rows to be merged
  • scope defines the td cells to which the th cell is linked (row/col)
